# Organiser
## Введение
В рамках учебной практики в НИТУ МИСИС была поставлена задача: разработать органайзер дя инструментов лаборатории FabLab с возможностью крепления на стену. Процесс подготовки включал себя несколько пунктов:
- поиск готовых решений на интернет-ресурсах (Pinterest)
- анализ и выбор решения для органайзера
- создание моделей-прототипов
- корректировка и улучшение моделей
- изготовление органайзеров

Также в процессе практики для освоения методов изготовления органайзеров были выполнены мини-задания.

Лазерная резка (ссылка на Git): https://github.com/ivanabbakumov/KIT

3Д-печать (ссылка на Git): https://github.com/ivanabbakumov/FFF-Tools

Помимо изготовления органайзеров было организовано рабочее пространство, вырезаны, отшлифоны и покрыты лаком сетки для рабочего пространства.

## Анализ и выбор решения для органайзера
Далее был проведен анализ органайзеров для каждого инструмента.
1. Гаечные ключи

<img width="456" height="428" alt="image" src="https://github.com/user-attachments/assets/9d758ca0-8924-4352-ad57-66ff3e313d5f" />

Первые два варианта более компактные, но в то же время более сложные в реализации и производстве по сравнению с номерами 3, 4. Также варианты 3 и 4 позволяют использовать пространство над гаечными ключами для других инструментов. Следовательно останавливаемся на варианте 3, его и будем изготавливать.

2. Термотрубки

<img width="466" height="412" alt="image" src="https://github.com/user-attachments/assets/3035a7f8-c5f8-40b2-9658-fd57c3b26b41" />

Наиболее технологичным и удобным представляется вариант 1 в виде диспенсера. Однако потенциально могут возникнуть проблемы с подачей трубок. Также их сложно разделить по типам так, как они отличаются по длине, ширине, цвету и т.д. В таком случае удобнее использовать органайзер 2 в виде «стаканчика».

3. Молотки и шпатели

<img width="470" height="424" alt="image" src="https://github.com/user-attachments/assets/4ad9e35f-2f90-40f1-a876-07c2941716aa" />

Видео по номером 3:

https://github.com/user-attachments/assets/e05cf782-2c2d-40e7-a4c6-bde7b6370af1

Для молотка и киянки отлично подходят органайзеры 1 и 2, однако сравнивая их органайзер 1 проще в реализации, однако менее удобен в использовании. Для стамесок и шпателей можно использовать органайзер 3, но необходимо оценить целесообразность в зависимости от количества подобных инструментов. В случае если подобных инструментов немного, можно использовать полочку органайзера для гаечных ключей. По итогу выбираем органайзер 1 и 3.
